# Training pipeline

## Full step

1. Design model (input ,outputsize, forward pass)
2. Construct loss and optimizer
3. Training loop
   + forward pass: compute prediction
   + backward pass:gradients
   + update weights

## list

+ Pytorch Model
+ Autograd
+ Pytorch Loss
+ pytorch optimizer (優化器)

+ super **linear regression**
